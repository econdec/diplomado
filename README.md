# Proyecto Diplomado

### 1. Clonar el Proyecto
'''bash
git clone https://github.com/econdec/diplomado.git
'''
### 2. Instalar dependencias
´´´bash
npm install
´´´

### 3. Copiar variables de entorno y configurar
´´´bash
cp .env.sample .env
´´´

### 4. Levantar el proyecto
#### 4.1 En desarollo
´´´bash
npm run dev
´´´

#### 4.1 En producción
´´´bash
npm run start
´´´
